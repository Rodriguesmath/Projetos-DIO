Introdução ao HTML

O HTML(Hypertext Markup Language) é uma linguagem de marcação e um dos elementos mais importantes da web. se comparamos um site com o corpo humano, o HTML seria o esqueleto e os elementos de as partes que compõe esse esqueleto, nesse viés, os atributos seriam o nome e as particularidades de cada parte. Ter essa compreensão do HTML, facilita entender a sua comunicação com o restante dos elementos que compõe a web no geral. A seguir, no projeto, abordaremos as tags apresentadas até aqui, obviamente, utilizado-as, para melhor exemplicar cada uma.

Tags de texto

Chamamos os elementos do HTML de Tags. Neste tópico, abordaremos as tags usadas para trabalhar especificamente com texto, tais quais: a, p, h1 a h6, hr, mark, small, i, sup, sub e blockquote. Confira abaixo:

  • <a>: É um elemento de ancoragem, usado para a criação de hyperlinks e atalhos para navegação em um site, como o utilizado para navegar nessa página.
  • <p>: sem dúvidas, a tag mais simples de se utilizar, ela cria um parágrafo de texto.
  • <h1> a <h6>: São tags de título variando em tamanho:

    h1

    h2

    h3

    h4

    h5
    h6
  • <hr>: Cria uma barra horizontal.
  • <mark>: Serve como um marca texto.
  • <small>: O texto escrito dentro desse elemento fica pequeno.
  • <i>: Texto em itálico
  • <strong>: Texto em Negrito
  • <sup>: Texto superescrito
  • <sub>: Texto Subscrito
  • <blockquote>:
    Cria um bloco de citação em meio ao texto.
  • <font>: Pode modificar tipo, cor e tamanho do texto.
  • <del>: Mostra o texto excluído de uma página
  • <abbr>: Define uma abreviação exemplo: EX.

Listas

As listas do HTML são de dois tipos: ordenada e não ordenada. Para melhor compreensão, mostraremos exemplos de cada um dos tipos:

  • Item da lista não ordenada 1:
  • Item da lista não ordenada 2
  • Item da lista não ordenada 3

A lista não ordenada, como o próprio nome diz, não possui ordem, possuindo apenas "bolinhas".

  1. Item da lista ordenada 1
  2. Item da lista ordenada 2
  3. Item da lista ordenada 3

A lista ordenada apresentada seus elementos ordenados de forma numerica.
